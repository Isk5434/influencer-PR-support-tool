# Instagramインフルエンサー向け化粧品PR × 薬機法AIチェッカー

企業やインフルエンサーが導入後を具体的に想像できる、プレミアムなデモUIです。

## 🎨 特徴

- **化粧品ブランドらしい上品なデザイン** - 淡いグラデーション、大きな余白、円形モチーフ
- **自動カルーセル** - 3つのサンプル投稿文が自動で切り替わる
- **インタラクティブなデモ** - 生成ボタン、コピー機能、トースト通知
- **薬機法配慮済みバッジ** - 安心感を演出
- **レスポンシブデザイン** - モバイルにも対応

## 🚀 使い方

### ローカルで実行

## ✨ 主要機能

### 1. ヒーローセクション
- 左側：サービスコンセプト説明
- 右側：スマホ画面風のAI生成デモ

### 2. 自動カルーセル
- 4秒ごとに投稿文が自動切り替え
- ホバーで一時停止

### 3. 生成ボタン
- クリックでローディングアニメーション
- 点滅エフェクトで「AIが考えている感」を演出

### 4. コピー機能
- ワンクリックで投稿文をコピー
- トースト通知で完了を表示

### 5. 生成例一覧
- 4つのカテゴリ（リップ、スキンケア、アイメイク、ベースメイク）
- 口調タイプ（フレンドリー、丁寧、カジュアル）
- リスクレベル表示
